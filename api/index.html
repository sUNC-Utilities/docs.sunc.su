sUNC Metadata API

The sUNC Metadata API provides machine-friendly JSON data for integrating sUNC function metadata into your projects. Choose between a compact Mini Version for lightweight usage or the Jumbo Version for complete library and function details.

Mini ±10KB

A simple function name to description dictionary, perfect for integrating basic metadata about sUNC functions into your application.

Access Mini Version

Jumbo ±45KB

A fully parsed and beautified dataset including library overviews, function signatures, parameter types, and links to the official documentation.

Access Jumbo Version

Both versions are served in application/json format and automatically update alongside sUNC documentation.
